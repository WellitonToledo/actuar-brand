# Wallpapers

Papéis de parede da marca Actuar — desktop e mobile.

_Em breve._ Adicione aqui os arquivos por resolução, ex.:

```
wallpapers/
├── desktop/   2560×1440, 1920×1080, 3440×1440 (ultrawide)
└── mobile/    1179×2556 (iPhone), 1080×2340 (Android)
```

Sugestão de nomenclatura: `actuar-wallpaper-<tema>-<largura>x<altura>.png`
(ex.: `actuar-wallpaper-azul-1920x1080.png`).
